make main을 입력하여 실행파일을 만든다.
./main 을 입력하여 실행한다

1.Create	
상품의 이름, 번호, 재고, 유통기한을 입력하여 상품의 레코드를 생성함

2.Read
상품의 이름을 입력하여 상품의 정보를 출력함

3.Update
상품의 이름을 입력하여 상품의 정보를 수정함

4.Delete
상품의 이름을 입력하여 상품의 레코드를 삭제함

5.List
전체상품의 목록을 출력함

6.Load file
members.txt파일의 내용을 레코드의 내용으로 불러옴

7.Write file
현재 레코드의 내용을 members.txt 파일에 저장함

8.Exdate_check
사용자의 현재시간으로부터 전체상품의 유통기한이 몇일 남았는지, 지났는지를 알려줌

9.Stock_check_Add
상품이름을 입력하여 재고를 확인하고  재고가 10개 미만인 상품은 재고를 추가할 수 있음

10.no_list
상품들을 상품번호순으로 정렬하여 리스트를 출력함
